## 3. How to Charge Them

### Payment Collection Options:

#### Option 1: PayPal Invoicing (EASIEST to start)

**Steps:**
1. Sign up for PayPal Business (free)
2. After salon agrees, send them a PayPal invoice
3. Set up recurring monthly billing
4. PayPal handles everything

**To send an invoice:**
1. Go to PayPal â†’ Send Money â†’ Create invoice
2. Add their email
3. Description: "Nail Art AI Featured Listing - [Tier Name]"
4. Amount: $29, $79, or $149
5. Set as recurring monthly
6. Send!

#### Option 2: Stripe Payment Links (PROFESSIONAL)

**Steps:**
1. Sign up for Stripe (stripe.com)
2. Create a "Payment Link" for each tier
3. Send the link to salon owners
4. Stripe handles subscriptions automatically

**Creating a Stripe Payment Link:**
1. Dashboard â†’ Payment Links â†’ Create
2. Product: "Featured Listing - Premium - $79/month"
3. Recurring: Monthly
4. Get link like: `https://buy.stripe.com/abc123`
5. Send link to salon owner

#### Option 3: Manual Invoicing (Works for first customers)

**Steps:**
1. Use free tools like Wave or FreshBooks
2. Create invoice with your logo
3. Include your PayPal/bank details
4. Send monthly invoices
